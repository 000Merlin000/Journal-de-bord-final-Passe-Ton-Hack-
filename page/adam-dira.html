PROBLÉMATIQUE 1 - Comment la programmation peut-elle resoudre les enigmes du Ce que je quotidien?

Description du challenge

Decodage Morse,le but de ce challenge était de trouver un code permettant de traduire n'importe quel texte morse

Étapes de résolution :

Creer un dictionnaire morse donnant chaque caractere morse avec a lettre le correspondant Traduire egalement les espaces entre lettres et mots enfin inserer dans une variable chaque lettre traduite pour former chaque mot puis inserer chaque mot pour former le texte final tarduit.

Outil utilisée : chat gpt

Mes difficultés recontrée :

Difficultes à separer les chaines de caractères. Difficultés à traduire les chaines de caractères morses en lettre de 'alphabet normal.

ce que j' ai appris :

  • l'usage de split(.), pour par exemple séparer tout les elements quand il se trouve un point
  • l'usage de get() pour recupérer les valeurs d'un dictionnaire à partind'une cle.

    je pourrais refléchir sur la manière dont j' ai résolu ce defie en repondant a la question posé : Comment la programmation peut-elle resoudre les enigmes du Ce que je quotidien?
